Reciprocal Function – Family Mart Rice Bowl

Transformations I chose this photo because me and my friends like to hangout a lot in family mart during our break from classes to get instant food. To model the bowls appearing in the photo, I began by identifying that it's shape closely resembled a Reciprocal function, So I selected the parent function Y=1/x^2 as a starting point which has two branches and a vertical asymptote. To better match the curved bowls in the photo, the graph was vertically compressed by changing from 1 to 0.01. This reduced the steepness of the curve so it aligned more perfectly with the shape of the bowl. Next, I translated the graph horizontally by 1.96 units to the right, this moved the horizontal asymptote from x=0 to x=1.96 and then I shifted the graph upward by 2.96 units to correctly position the curve over the image. Finally, A restricted domain was applied because only a portion of the reciprocal squared function matches the visible section of the bowl in the photo. These transformations helped create a realistic mathematical model of the object.
